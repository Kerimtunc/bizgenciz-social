# Task ID: 3
# Title: FAZ 3: API Mimarisi Tasarımı ve Endpoint Grupları
# Status: done
# Dependencies: 2
# Priority: high
# Description: Multi-tenant API middleware, B2B/B2C API endpoint grupları, Ciro Partnerliği API'leri, kurye yönetimi API'leri ve modern API standartları
# Details:
Bu faz, projenin API altyapısını oluşturacak kapsamlı endpoint tasarımını içerir. Multi-tenant API middleware (tenant validation, tenant isolation, feature flag, rate limiting, audit logging) geliştirilecek. B2B API endpoint grupları (dashboard, orders, menu management, customer management, staff management, kitchen management, table management, reservation management, notification system, business management, analytics and reporting, system management) tasarlanacak. B2C API endpoint grupları (restaurant discovery, menu, order flow, user profile, credit system, location services, notification) oluşturulacak. Ciro Partnerliği API'leri (credit management, subsidy management, commission, loyalty program) ve kurye yönetimi API'leri (courier registration, courier assignment, courier performance, courier communication, courier payment) tasarlanacak. Modern API standartları (idempotency, asenkron işlem yönetimi, global health check, API dokümantasyonu, error handling, rate limiting) uygulanacak. Süperadmin API endpoint'leri (tenant management, feature management, feature assignment, bulk operations, analytics, system management, template management, audit logs) geliştirilecek.

# Test Strategy:
Her API endpoint için Swagger/OpenAPI 3.0 dokümantasyonu oluşturulacak. Unit testler yazılacak ve integration testleri yapılacak. Performance testleri gerçekleştirilecek. Eksiksiz veriye erişmek için @insaedilecekler.md dosyasından faydalanılacak ve buna rağmen orada eksik kalan özellikler tespit edilecek. Eksiklik tespiti 'QR Menü, POS sistemi, Yemek Sipariş Uygulaması ve Çoklu Tenant ile toplu olarak restoran ve kafelere hizmet vermek isteyen bir şirket için' kapsamında yapılacak. Tespit edilen eksiklikler için detaylı çözüm önerileri ve implementasyon planları hazırlanacak.
